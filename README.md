# 🛡️ Cyber Safety Quiz Awareness Platform

## 📌 Project Overview
The Cyber Safety Quiz Awareness Platform is a web-based application designed to spread awareness about cyber security through interactive quizzes. It helps users learn about online threats, safe practices, and digital responsibility in an engaging way.

## 🎯 Objectives
- Promote cyber safety awareness
- Educate users about online threats (phishing, hacking, scams)
- Provide an interactive quiz-based learning experience
- Track user performance and improvement

## 🚀 Features
- 👤 User Registration & Login System  
- 🧠 Interactive Cyber Safety Quiz  
- 📊 Score Tracking Dashboard  
- 🔐 Secure Authentication System  
- 📝 Quiz Attempt & Review System  
- 🛠️ Admin Panel for managing quizzes  
- 🚨 Security Violation Logging  

## 🏗️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: Apache (XAMPP/WAMP)

## ⚙️ Installation & Setup

1. Clone the Repository
git clone https://github.com/your-username/cyber-safety-quiz.git

2. Move to Server Directory
- Copy project folder to htdocs (XAMPP) or www (WAMP)

3. Setup Database
- Open phpMyAdmin
- Create database (cyber_quiz)
- Import SQL file

4. Run the Project
- Start Apache & MySQL
- Open browser:
http://localhost/cyber-safety-quiz
